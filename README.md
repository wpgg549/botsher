# ANT BOT FINAL

Bot WhatsApp share teks + media (Baileys)

## Install (Termux)

pkg install nodejs git
git clone https://github.com/wpgg549/botsher.git
cd antbot
npm install
node index.js

## Cara Pakai

1. Kirim teks / gambar
2. Reply:

!share → teks aja
!shareall → semua (gambar/video)

## Note

- Jangan spam
- Delay sudah random (15–25 detik)
